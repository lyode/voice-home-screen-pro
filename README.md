### **Transparency Tips**
- **0-20%**: Very transparent (best for light backgrounds)
- **20-40%**: Balanced (recommended)
- **40-60%**: Semi-opaque
- **60-100%**: Solid (blocks background)

### **Blur Settings**
- **0-10px**: Sharp, minimal blur
- **10-25px**: Glassmorphism effect (recommended)
- **25-50px**: Heavy blur, dream-like
